# Glug & Poop UI 超简版规格

## 页面命名

- 首页 `Home`
- 喝水记录页 `Water Log Sheet`
- 饮品记录页 `Drink Log Sheet`
- 食物记录页 `Food Log Sheet`
- 便便记录页 `Poop Log Sheet`
- 今日总览页 `Dashboard`
- 日历页 `Calendar`
- 日历记录区 `Calendar Records Panel`
- 日历记录卡片 `Calendar Record Card`

## 整体风格

- 偏 iOS 原生，但更活泼一点
- 大圆角、粗标题、浅阴影
- 白底为主，功能卡片用高饱和主题色
- 不要拥挤，留白要舒服
- 交互以底部弹层和卡片为核心

## 主色

- 黑色 `#1C1C1E`
- 浅灰 `#F2F2F7`
- 水 `#007AFF`
- 饮品 `#AF52DE`
- 食物 `#FF2D55`
- 便便 `#FFCC00`

## 字体层级

- 首页主标题：`34 / Black / Rounded`
- 弹层标题：`26 / Black / Rounded`
- 重点数值：`28-30 / Black / Rounded`
- 按钮文字：`17 / Bold`
- 分区标题：`13 / Bold`
- 小标签：`11-12 / Bold`
- 说明文字：`12-14 / Medium/Semibold`

## 圆角和间距

- 大卡片圆角：`32`
- 弹层圆角：`28`
- 中卡片圆角：`22-24`
- 小组件圆角：`12-18`
- 胶囊按钮：`100`
- 页面左右边距：`24`
- 大模块间距：`20-30`
- 弹层内部间距：`16`

## 首页

- 顶部：主标题 + 副标题
- 中间：2x2 功能入口卡片
- 底部：`Dashboard` 和 `Calendar` 两个按钮
- 功能入口卡片要大、彩色、1:1 比例、图标明显

## 喝水记录页

- 顶部标题：`HYDRATE!`
- 上方一排快捷水量按钮：`100 / 300 / 500 / 750 / 1000ml`
- 中间是蓝色水量选择面板
- 下方是时间、备注、提交按钮

水量面板重点：

- 数值范围 `0-1500ml`
- 左右滑动选择
- 中间数值最突出
- 两侧刻度逐渐透明
- 蓝色背景里显示 `0ml` 和 `1500ml`
- 波形贴底，不要悬空

## 饮品记录页

- 标题：`CHOOSE POISON`
- 2 列按钮网格
- 选中态用饮品紫色
- 下方保留时间、备注、照片、提交

## 食物记录页

- 标题：`FEED ME`
- 2 列按钮网格
- 选中态用食物粉色
- 比饮品页多一个 `Duration`

## 便便记录页

- 标题：`CAPTAIN'S LOG`
- 上方：便便形状，4 列网格
- 下方：便便颜色，2 行 4 列
- 选中态用便便主题色，不要黑色
- 继续保留时间、时长、备注

## Dashboard

- 顶部返回按钮
- 标题：`TODAY!`
- 中间是今天记录的大卡片列表
- 底部悬浮 AI 文案卡片
- 卡片可点击编辑

卡片里可出现：

- 类型图标
- 时间
- 内容标签
- 时长
- 备注
- 图片

## Calendar

- 顶部：返回 + 月份切换
- 中间：月历格子
- 下方：当天记录展示区
- 点击日期时，只切换下方内容，不弹新卡片

下方记录区支持：

- 左右滑切日期
- 上下滑看记录

## 日历记录区

- 顶部显示：
  - `RECORDS`
  - 当前日期
  - 数量 badge
- 中间一排四个新增按钮
- 下方是当天记录卡片区

## 日历记录卡片

- 双列布局
- 自适应内容高度
- 右下角可放图片缩略图
- 卡片背景用对应类型颜色的浅色版本
- 不要出现大块空白
